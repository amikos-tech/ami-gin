## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| developer branch → GitHub Actions runner | Untrusted repository contents and workflow definitions execute on hosted runners. |
| GitHub Actions runner → GitHub Actions artifacts and summaries | Test outputs and coverage summaries are stored and rendered inside GitHub-managed services. |
| GitHub Actions runner → GitHub checks API | Check results determine whether protected branches can be updated. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-04 | I | GitHub-native coverage reporting | mitigate | Keep CI-04 inside GitHub-managed artifacts and `$GITHUB_STEP_SUMMARY`; do not introduce Codecov secrets, tokens, or third-party reporting in this phase. |
