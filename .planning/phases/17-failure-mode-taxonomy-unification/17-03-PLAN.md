<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Caller JSON bytes -> parser/staging | Untrusted document content can fail parsing, transformer conversion, or numeric classification. |
| Staged document state -> durable builder indexes | Soft skips must return before staged observations mutate shared path/index/bookkeeping state. |
| Internal merge invariant -> tragic builder state | Validator-missed merge panics must close the builder rather than being treated as caller-configured soft skips. |
