| env vars -> tier activation | silent activation or silent fallback would make runs irreproducible |
| corpus root path -> benchmark setup | a bad external path must fail closed when the user explicitly opted in |

| T-11-03 | T | external root path | mitigate | Fail with a precise `GIN_PHASE11_GITHUB_ARCHIVE_ROOT` layout message when an enabled tier cannot find shards. |
