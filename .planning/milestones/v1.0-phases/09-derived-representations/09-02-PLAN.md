<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| raw path lookup -> alias lookup | ambiguous routing can silently query the wrong representation |
| in-memory representation metadata -> encoded bytes | missing or inferred alias metadata can break decode parity |
| opaque custom companion -> encode/decode contract | pretending to serialize unreconstructible functions would corrupt user expectations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-04 | T/I | alias query routing | mitigate | Require `gin.As(alias, value)` for companion queries and keep raw-path queries default. |
| T-09-05 | T/R | representation metadata | mitigate | Persist explicit representation entries and rebuild alias lookup from them on decode. |
| T-09-06 | R | custom companion serialization | mitigate | Reject `Serializable: false` companions during `Encode()` instead of writing partial or misleading metadata. |
</threat_model>
